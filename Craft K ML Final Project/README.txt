This code was created in PyCharm in a virtual environment (venv). The necessary packages that need to be installed prior to running the code are pandas, numpy, keras, tensorflow, matplotlib, and scikit-learn.

If the environment does not have the packages already installed, either use pip install pandas numpy keras tensorflow matplotlib scikit-learn in the terminal, or add the packages manually. In PyCharm I did file>settings>+ icon next to package and added them manually.

To run the code, open the code file folder titled "Craft K ML Final Project" and use the virtual environment interpreter. Proceed to run the "data_proc" file first, followed by "satellite_if", "vae", "lstm", then "aucroc".
